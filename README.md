# 衫云智管 - 静态演示版

> 服装零售 AI 多端智能管理系统 v4.0

## 快速启动

```bash
# 方式一：一键启动
./start.sh

# 方式二：手动启动
python3 -m http.server 8765

# 方式三：直接打开
# 双击 index.html 用浏览器打开
```

访问 http://localhost:8765 ，使用 **demo / demo123** 登录。

## 功能特性

### 核心功能
- **多门店管理** - 支持连锁店独立运营
- **进销存管理** - 采购、销售、库存一体化
- **移动开单** - 触屏优化，支持扫码
- **客户管理** - 会员积分、消费记录

### v4.0 新增功能
- **现金流看板** - 今日实收/本月实收/应收/应付
- **退货管理** - 一键退货，自动回补库存和积分
- **积分闭环** - 100积分=1元，结账时抵扣
- **角色权限** - boss/manager/clerk 三级权限
- **库存盘点** - 盘点录入、差异报告、应用模式
- **员工管理** - CRUD 员工账号和角色

### AI 能力
- **AI 智能决策** - 智能推荐、智能定价
- **营销文案生成** - 一键生成促销文案
- **数据分析洞察** - 销售报表、库存预警

## 技术架构

```
单文件架构：index.html + app.js + api.js + api-mock.js
数据存储：localStorage (浏览器本地)
无需后端：纯前端即可运行完整功能
```

## 部署

静态文件可部署到任意 Web 服务器或 CDN：
- GitHub Pages
- Vercel / Netlify
- 阿里云 OSS / CDN
- Nginx / Apache
